# Obss-internship-project,


* My internship project for OBSS. It's a full stack web application. Details can be found in [this report](/dokumanlar/sunum.docx).

* Backend code is in [src](/src) folder. It's a Spring Boot application written in Java 11.

* Frontend code is in [frontend](/frontend) folder. It's a React application written.

* PostgreSQL is used as database.

* Spring application has a Javadoc and a Swagger documentation. Thanks to SonarQube, it has a static code analysis.

* Application is dockerized and has a Jenkins pipeline.
